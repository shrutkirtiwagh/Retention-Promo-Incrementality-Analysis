## Measuring Retention Promo Incrementality

**Tools:** Python, SQL (SQLite), Pandas  
**Skills:** Churn analysis, at-risk segmentation, holdout test design, incremental lift measurement

Streaming platforms risk cannibalizing revenue by sending retention promos to users who 
would have stayed anyway. This project identifies at-risk subscribers using SQL-based 
behavioral segmentation, then designs a randomized holdout test to isolate true 
promo-driven renewals from organic ones.
